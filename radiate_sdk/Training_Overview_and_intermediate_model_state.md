# Training Overview and Intermediate Model State

## Training Overview

- Train the HydraFusion model (stems + branches) for object detection.
- Train the Gating Module using outputs from the trained HydraFusion model.
- Fine-tune the Fusion Block to integrate multi-modal predictions.
